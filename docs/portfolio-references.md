## Decisões para este portfolio

- Manter 5 projetos fortes, na ordem definida pelo usuário.
- Abrir com posicionamento claro: Full Stack, IA, automação, APIs e operação real.
- Cada projeto explica produto, stack, agentes/processos, banco/infra e responsabilidades.
- Visual preto com verde como detalhe, evitando aparência genérica de currículo.
- Usar prints e vídeo onde são seguros; onde há dados sensíveis ou faltam imagens, usar placeholder redigido.
- Não expor IP, credenciais, chaves, dados de clientes ou prints crus de CRM.
